# vue_kutx_playlist_explorer

[kutx.netlify.app](https://kutx.netlify.app)

A mobile friendly web app built on the JAMstack, designed to stream Austin's KUTX radio live while updating the playlist in real time.
***

### Built With: 

Bootstrap 4

Vue 2

KUTX's API
***

npm install
